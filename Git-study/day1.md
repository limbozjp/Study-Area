### 基础指令学习
```
- cd ~                  回到home
- cd [foldername]       进入目录
- cd ..                 回退上一层目录
- pwd                   显示完整路径
- touch [filename]      创建文件
- mkdir [foldername]    创建文件夹
- rm [filename]         删除文件
- rm -r [foldername]    删除文件夹
- mv [filename] src [foldername]        将同一目录下的文件移动到某个文件夹下
- open -e [filename]    打开文件
- cat [filename]        在终端显示文件内容
- echo [text] > [filename]              写入内容到文件
  echo [text] >> [filename]             追加内容到文件
- reset                 重新初始化终端/清屏
- clear                 清屏
- history               查看命令历史
- help                  帮助
- exit                  退出
- #                     表示注释    echo hello world > demo.txt #你好，世界
```
